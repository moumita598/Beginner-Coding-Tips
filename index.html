Beginner’s Coding Tips

1. Start it

Don’t build Facebook on day one. Make a button. Click it. Break it. Fix it. Repeat ittttttttttt

2. Google OG will solve your all doubts

Typing: “why won’t my CSS center” is a rite of passage. StackOverflow is love.

3. Understand, Don’t Memorize

If a loop feels like magic, don’t skip it. Understanding it do not Copy Paste Power

4. Code Every Day 10 mins atleast

Momentum is better than motivation. Just one line today is a win
